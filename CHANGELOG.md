# Changelog

本项目的用户可感知变化记录在此。产品裁决与架构差异请见 `docs/CHANGES.md`。

## [Unreleased]

### Fixed

- 启动时会对“已结束但尚未写入历史”的会话做幂等对账，避免进程恰好中断在分文件持久化之间时丢失记录。
- 本地 JSON 读取、导入和写入增加 64 MiB 硬上限，并限制会话数量、文本字段与连续中止历史，避免异常文件导致启动卡死或内存耗尽。
- 超过七天的睡眠或挂起会按每个状态的自然时间边界追赶，不再把冷却期从唤醒时重新计时。
- 系统日期回拨不再刷新每日额度；剩余时间始终限制在已配置会话长度内。
- 系统日期回拨后的冷启动也会保留已使用额度，直到真实日期越过持久化日键。
- 活动会话期间锁定节律与额度设置，避免中途修改重新解释当前计时。
- 假数据生成器与 token 格式化现在安全处理非有限浮点数和整数极值。
- 历史接近数量或 64 MiB 上限时会精确淘汰最旧前缀并保留最新会话；跨文件写失败不会再越过 sessions 去覆盖恢复快照。
- 本地文本同时受字符数和 UTF-8 字节数约束，组合字符无法绕过资源上限。
- 关闭八天后重启仍可从合法快照追赶并补回未完成会话；异常时钟中止不再触发浮点转整数崩溃。

### Changed

- 重做应用图标与社交预览：移除卡通拟人和仿仪表盘元素，统一为更克制的几何番茄计时器视觉。
- 打包 App 禁止多实例，并内置应用图标、项目许可证及锁定依赖的第三方许可声明。
- GitHub Actions 固定到完整提交 SHA；公开树增加隐私、品牌中立、许可和工作流供应链门禁。
- 项目转为源码优先的开源发布：提供社区治理文件、双语详情页、真实应用截图和原创品牌素材；在完成 Apple 公证前不提供公开二进制下载。

## [3.2.2] - 2026-07-14 (private pre-release)

### Fixed

- 首次确认免责声明前不再写入或注册默认全局快捷键；确认后只安装一次。
- 修复九状态/三主题快照装配被免责门拦截而重复渲染 idle/provider A 的假阳性。
- 拒绝超出 `0...1440` 分钟的运行快照，并防御完成页毫秒换算溢出。
- 跨午夜后第一次发起请求会立即刷新昨日额度，无需等待后台 ticker。
- 清除安全 smoke 留下的临时 LaunchServices `rlt:` 注册，并验证精确路径与临时目录均已消失。
- manifest、SHA、公证 App 的多文件提升改为可回滚事务；信号与清理失败不再假成功。

### Changed

- UI 中剩余的 API 风格标签、协议文案、热力图图例/月标和开发版占位统一经过 i18n catalog。
- 通知中心改为可注入边界，授权时机、类别与 payload 可在不访问 TCC 的测试中验证。
- Reduce Motion 覆盖折叠码块、RESET 数字变化与热力图 hover。
- 文档提交不再消耗完整 GitHub macOS CI 矩阵；tag 仍始终执行发布门禁。

## [3.2.1] - 2026-07-14 (private pre-release)

### Fixed

- 首次免责确认将覆盖菜单、快捷键、URL、通知和计时器等全部状态变更入口。
- 系统通知权限延后到免责确认后请求；确认前设置页第三方控件保持只读禁用。
- 持久化不可用时不再把随机临时目录伪装为可靠存储。
- 加固每日额度、引擎快照、跨午夜统计、418 连击和导入失败语义。
- 消除已知 Swift 并发隔离警告，固定法律文案双语显示，并让设置页版本读取 App Bundle。

### Changed

- 发布链增加版本一致性、红线、资源 bundle、架构、签名和归档校验。
- GitHub Actions 在 macOS 15 arm64 跑完整门禁、在 macOS 15 Intel 跑非渲染门禁，并把 universal 候选下载到 macOS 14 arm64 做最低系统完整性与启动 smoke。
- 运行时与公开发行文案保持品牌中立；公开规格仅保留可复验的产品与工程事实。

## [3.2.0] - 2026-07-08

- 完成三模块 SwiftPM 架构、菜单栏 App、JSON 本地存储、双层热力图、通知、全局快捷键、URL scheme 与 Provider A/B/C 主题。
- 完成设置页、持久化、快照和测试基建的多轮加固。
